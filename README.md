# swiss-albanian-network-website
Statische, responsive Vereinswebsite für das Swiss Albanian Network (SAN) mit einheitlicher Navigation (inkl. Hamburger-Menü), Home/Über den Verein/Vorteile sowie einer Event-Übersicht mit einzelnen Veranstaltungsseiten. Enthält außerdem Impressum und Datenschutz, optimiert für Desktop, Tablet und Smartphone.
